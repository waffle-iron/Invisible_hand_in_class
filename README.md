# Invisible_hand_in_class

## Development Environment
* OS
  * Ubuntu 16.04.2 LTS

## Role
* file
  * function.c - 서버와 클라이언트에서 사용하는 함수구현 
  * server.c - server-side programming
  * client.c - client-side programming

* compile시

```c
make          // 서버 + 클라이언트 
make server   // 서버
make client   // 클라이언트만 
make clean    // exe파일,object파일 삭제 시
```
